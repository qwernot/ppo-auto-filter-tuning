# Specifications

This directory contains JSON files used by the tuning and training scripts.

- `filter_example_one_target.json`: electrical target definition.
- `filter_example_one_tuning_manifest_lowcost_fixheight.json`: low-cost five-variable tuning manifest.
- `filter_example_one_ppo_train_config_fixheight_v2.json`: PPO training and evaluation configuration.
- `filter_example_one_reference_tuning_manifest_v2_highres_fixheight.json`: high-resolution review manifest.
- `filter_example_one_reference_anchor_fixheight_v1.json`: reference low-cost anchor metrics.
- `filter_example_one_reference_highres_review_fixheight_v1.json`: reference high-resolution review metrics.

The default script entry points use the low-cost tuning manifest and PPO configuration.
